## Installation
```bash
git clone https://github.com/Innov8or123/Research-Papers-Website.git
cd Research-Papers-Website
npm install
```
---

## Configuration
- Edit config.js for MySQL credentials.
- Example:
  ```.env
  DB_HOST=localhost
  DB_USER=your_user
  DB_PASSWORD=your_pass
  DB_NAME=research_papers
  ```
